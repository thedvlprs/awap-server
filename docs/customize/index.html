Customize this API

You can download our API source code (github.com/aleoheen/awap-server), modify it and host at any address. In this page you'll see installation instructions.

What you need to to this

  • Linux machine;
  • Node.JS installed (with NPM);
  • MongoDB installed and runned

How to install

  1. Download our API source code (github.com/aleoheen/awap-server)
  2. Go to the project folder: cd awap-server
  3. Edit config file (./core/config.js) as you want.
  4. Install nodemon for run in development mode (sudo npm i -g nodemon)
  5. Install dependencies (npm i)

How to run

In production mode: yarn run start In development mode: yarn run debug
